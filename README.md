# pay_mena Ticket Reporter

Tampermonkey-скрипт для автоматического сбора и выгрузки тикетов в XLSX.

## Установка

1. Установи расширение [Tampermonkey](https://www.tampermonkey.net/)
2. Нажми [👉 Установить скрипт](https://raw.githubusercontent.com/ТВОй_ЛОГИН/pay-mena-reporter/main/pay_mena_reporter_v7.user.js)

## Использование

- Открой страницу с тикетами на th-managment.com
- Выбери агента и период в панели справа внизу
- Нажми «Запустить сбор»
